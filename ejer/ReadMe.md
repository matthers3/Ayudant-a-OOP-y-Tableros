Enunciado
========================

Se realizara un ejercicio de tableros que deba contener lo siguiente.


Dados
=====

Se debe programar la clase dado, la cual debe recibir una cantidad fija de caras, debe ser capaz de ser lanzado y retornar un valor.


Aventurero
==========

Nuestro personaje, debe tener nombre, cuantos años lleva en el calabozo, y debe ser capaz de portar varios dados.
El aventurero debe poder tirar todos sus dados a la vez.


Tablero
=======

Esta clase debe poseer un tablero como lista de listas y debe tener métodos para mover aventureros y otros objetos dentro del tablero.
También se debe hacer un método para encontrar la posición de algún elemento dentro del tablero.
