Enunciado
========================

Se realizara un ejercicio para comprender mejor el uso de objetos y de tableros, para esto debes programar el sistema de movimiento de aventureros dentro de un calabozo.


Dados
=====

Se debe programar la clase dado, la cual debe:

- Tener una cantidad de caras entregadas al ser creado el dado. 
- Ser capaz de ser lanzado y retornar un valor como un numero entero.


Aventurero
==========

Nuestro personaje, debe tener:

- Nombre 
- Cuantos años lleva en el calabozo
- Debe ser capaz de portar varios dados y lanzarlos todos a la vez.
- Al ser printeado debe ser representado con la primera letra de su nombre en mayuscula.


Tablero
=======

Esta clase debe poseer:

- Un tablero como lista de listas.
- Poder representar el tablero en la consola.
- Tener métodos para mover aventureros y otros objetos dentro del tablero.
- Debe tener un método para encontrar la posición de algún elemento dentro del tablero.
- Debe marcar las posiciones a las que se puede mover un aventurero.
- Debe corroborar, a partir de las marcas, que el movimiento es válido.
